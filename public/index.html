Timestamp API

How to use:

  • Pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  • If it does, it returns both the Unix timestamp and the natural language form of that date in JSON format.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example:

  • Input:
    • https://fcc-pitlv2109-timestamp.com/December 15, 2015
    • https://fcc-pitlv2109-timestamp.com/1450137600
  • Output:
    • { "unix": 1450137600, "natural": "December 15, 2015" }
